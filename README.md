# MusabMine OS (MMOS)

Enterprise-grade, AI-first, plugin-based mining operating system.

## Repository Structure

- `apps/` — Deployable applications
- `packages/` — Shared libraries
- `services/` — Independent services
- `infrastructure/` — Infrastructure as Code
- `docker/` — Container assets
- `scripts/` — Automation scripts
- `docs/` — Operational documentation
- `tools/` — Internal developer tools

## Development Principles

- Documentation First
- Plugin-Based Architecture
- Universal Miner Compatibility
- Enterprise Coding Standards
- AI-First Design
- Docker-First Development

Refer to the frozen MMOS architecture and implementation guides for detailed design decisions.
